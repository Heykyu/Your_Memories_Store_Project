# Your_Memories_Store_Project

### Project Description

- 현재 운영중인 여러 SNS를 보고 제작하게 되었음. 최대한 사용자가 이해하기 쉽게 UI를 구성 하였고 입력 누락된 부분이 있을시 툴팁으로 알려줌. 무분별한 회원가입을 막기위해 gmail 프레임워크를 이용하여 회원가입 시 입력한 이메일로 유저 인증메일이 전송하게 구현.

### Usage Framework
